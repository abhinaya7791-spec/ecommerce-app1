# E-Commerce Web App

A full-stack ecommerce web application built using:

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB

## Features

- User Login & Register
- Product Listing
- Add to Cart
- Order Management
- Admin Dashboard

## Installation

### Client

```bash
cd client
npm install
npm start
```

### Server

```bash
cd server
npm install
npm run dev
```
